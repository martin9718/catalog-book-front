# Catalog Book

## Project setup

### Have node installed

### Have vue installed

```
npm install vue
```

### Have vue cli installed

```
npm install -g @vue/cli
```

### Rebuild node_modules

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
